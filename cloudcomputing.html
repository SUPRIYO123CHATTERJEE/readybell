About The Program

Drive your career forward with this industry-ready Cloud professional program to become a sought after Cloud Administrator to maintain the infrastructure and functionality of a company's cloud structures or a Cloud Solution Architect to implement and oversee a company's cloud computing strategy. This program starts with a Cloud Fundamental curriculum to ensure the participants first learn the cloud basics. The program the following 3 Cloud Computing training certification level program:

  • AZ-900 Microsoft Azure Fundamentals.
  • AZ-104 Microsoft Azure Administrator.
  • AWS Certified Solution Architect Associate.
Key Skills developed

Computing fundamentals, implement, monitor, and maintain Microsoft Azure solutions, lead architectural design sessions, develop proof of concepts/pilots, implement projects, and deliver ongoing refinement and enhancement

Program Outcome:

Post completing all theories and hands-on assignments students will be able to:

  • Implement, manage and monitor identity, governance, storage, compute, and virtual networks in a cloud environmen
  • Develop, design, deploy, migrate, integrate, configure, secure an Azure infrastructure
  • Determining workloads requirements & running workloads securely on Azure
  • Design of cost and performance optimized solutions, demonstrating a strong understanding of the AWS Well-Architected Framework.
Job Opportunities:
  • Cloud Engineer
  • Cloud Administrator
  • Cloud Architect

Program Curriculum

MS AZURE AZ-900 Microsoft Azure Fundamentals

Duration: 30 Hrs

Describe cloud computing, Describe the benefits of using cloud services, Describe cloud service types, Describe the core architectural components of Azure, Describe Azure compute and networking services, Describe Azure storage services, Describe Azure identity, access, and security, Describe cost management in Azure, Describe features and tools in Azure for governance and compliance, Describe features and tools for managing and deploying Azure resources, Describe monitoring tools in Azure

MS AZURE AZ-104 Microsoft Azure Administrator

Duration: 50 Hrs

Configure: Azure Active Directory, user and group accounts, subscriptions, Azure Policy, role-based access control, Azure resources with tools, use Azure Resource Manager, resources with Azure Resource Manager templates, virtual networks, network security groups, Azure Firewall, Azure DNS, virtual network peering, VPN Gateway, ExpressRoute and Virtual WAN, network routing and endpoints, Azure Load Balancer, Azure Application Gateway, storage accounts, Azure Blob Storage, Azure Storage security, Azure Files and Azure File Sync, Azure Storage with tools, virtual machines, virtual machine availability, virtual machine extensions, app service plans, Azure App Services, Azure Container Instances, Azure Kubernetes Service, file and folder backups, virtual machine backups, Azure Monitor, Azure alerts, Log Analytics, Network Watcher

AWS Certified Solution Architect Associate

Duration: 30 Hrs

Introduction to Cloud Computing, AWS Global Infrastructure, Amazon VPC (Virtual Private Cloud), Amazon EC2 (Elastic Compute Cloud), Amazon EC2 Auto Scaling, Amazon EBS (Elastic Block Storage), Amazon S3 (Simple Storage Service), Amazon EFS (Elastic File System), Amazon FSx for Windows File Server, Elastic Load Balancing (ELB), AWS Certificate Manager, AWS IAM (Identity and Access Management), Amazon RDS (Relational Database Service), Amazon DynamoDB, Amazon Route 53, Amazon CloudFront, AWS CloudFormation, AWS Lambda, Amazon EventBridge, Amazon SNS (Simple Notification Service), Amazon CloudWatch, Amazon CloudTrail
